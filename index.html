Divs e Span

As divs são do tipo block level e não tem significado semântico. É utilizada para agrupar elementos ou estilizar elementos. Usamos div quando não temos uma outra Tag com um significado semântico para colocar.

Abaixo temos um exemplo de como utilizar uma div:

Já a Tag span é um elemento inline level e serve para englobar e estilizar um outro elemento. O span também pode ter uma class
